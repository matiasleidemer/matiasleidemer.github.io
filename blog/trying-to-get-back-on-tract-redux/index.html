Trying to get back on tract (redux)
Sep 19, 2016
1 minute read

It has been a while since I blogged. For some reason I just can’t find the time, or motivation, to start writing something useful. More than once it feels like I have nothing to say at all.

I found a lot of excuses to not write. My favorite is not being able to find a good blog engine. I’ve been searching for quite some time for the perfect one.

  • Jekyll is awesome, but I don’t wanna spend time tailoring it, but I also don’t like any layout I found out there;
  • The same happens for Hexo;
  • Wordpress never sounded appealing to me;
  • Medium seems like a good option, but for some reason I don’t feel like writing code stuff there;

But I finally came across Gohugo and its beautiful cocoa theme. I guess my excuses are gone for real. No more messing around, let’s get this started.

I will redirect leidemer.com to this repository and it will be my main website.

Onward!
